Lexiflux

Version {{ version }}

A reading application that supports EPUB, HTML, plain text, and web page imports. Click any word or phrase to get instant translations, explanations, etymology, and AI-generated insights using various language models (OpenAI, Claude, Google, etc.).

Features include customizable sidebar with AI insights, vocabulary tracking with Anki export, reading progress synchronization, and multi-language support.
