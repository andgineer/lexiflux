How to Import Books from Calibre

You can send books directly from Calibre to Lexiflux using a simple plugin.

Don't have Calibre? It's a free e-book management application - download it here.
Setup Instructions:
  1. Download Lexiflux Plugin: Download Plugin
  2. Install: Calibre → Preferences → Plugins → Load plugin from file
  3. Restart: Restart Calibre after installation
  4. Use: Select books → Right-click → "Send to Lexiflux" or keyboard shortcut Cmd+Shift+L (Mac) / Ctrl+Shift+L (Windows/Linux)
  5. Alternative: Check Calibre's main menu bar or toolbar for "Send to Lexiflux" option
Supported Formats:
EPUB HTML TXT
